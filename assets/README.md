# Assets Directory

This directory should contain the app icon and splash screen images.

## Required Assets

- `icon.png` - App icon (1024x1024px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `splash.png` - Splash screen (1242x2436px)
- `favicon.png` - Web favicon (48x48px or larger)

## Temporary Solution

For development, the app will run without these assets, but you may see warnings.
You can use placeholder images or create simple ones using any image editor.

## Quick Placeholders

You can create simple placeholders using online tools or design software, or the app will use default Expo placeholders during development.
